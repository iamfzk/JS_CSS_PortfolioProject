## üöÄ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A text editor (VS Code, Sublime Text, etc.)

### Installation

1. **Clone or download** this repository
2. **Open** `index.html` in your web browser
3. **That's it!** The website is ready to use

### Customization

#### 1. Add Your Profile Image
- Replace `img/hero.jpg` with your professional photo
- See `IMAGE_SETUP.md` for detailed instructions and AI image generation tips

#### 2. Update Contact Form
- The contact form currently shows an alert on submission
- To make it functional, you can:
  - Use a service like [Formspree](https://formspree.io/)
  - Use [EmailJS](https://www.emailjs.com/)
  - Connect to your backend API

#### 3. Update Social Links
- All social media links are already configured
- Verify they point to your correct profiles

#### 4. Customize Colors
- Edit CSS variables in `styles/styles.css`:
  ```css
  :root {
    --color-secondary: #ff6b6b; /* Change this to your brand color */
  }
  ```
